## Build

### Release

```bash
cmake -B build . -DCMAKE_BUILD_TYPE=Release
```

generator Ninja:

```bash
cmake -G Ninja -B build . -DCMAKE_BUILD_TYPE=Release
```

### Debug

```bash
cmake -B build . -DCMAKE_BUILD_TYPE=Debug
```

generator Ninja:

```bash
cmake -G Ninja -B build . -DCMAKE_BUILD_TYPE=Debug
```
